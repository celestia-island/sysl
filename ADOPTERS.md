## How to Add Your Project

1. Fork the [sysl](https://github.com/celestia-island/sysl) repository.
2. Add your project to the table above, maintaining alphabetical order.
3. Submit a pull request with the title `Add [Project Name] to adopters`.
4. By adding your project, you affirm that your project is licensed under SySL.

## Criteria

Projects listed here should:

- Have a valid `LICENSE.txt` or `LICENSE` file containing the SySL text.
- Have a valid `AI_DISCLOSURE` file describing AI usage.
- Be publicly accessible (open source or source-available).

The SySL maintainer reserves the right to remove entries that do not comply.
