## Contributing Translations

### For New Languages

1. Create a directory under `l10n/` using the [BCP 47](https://tools.ietf.org/html/bcp47) language tag (e.g., `l10n/it/` for Italian).
2. Copy and translate the following files:
   - `LICENSE.txt` — the full license text
   - `README.md` — the project README
   - `FAQ.md` — the frequently asked questions
3. Submit a pull request.

### For Existing Languages

If you find an error or ambiguity in an existing translation, please:

1. Open an issue describing the problem.
2. If possible, include a suggested correction.
3. Translations are verified against new SySL versions by the maintainer.

### Quality Guidelines

- Legal terms should be translated conservatively. Prefer accuracy over elegance.
- The word "Author" (capitalized, as defined in Section 1 of the license) should be translated consistently within each language.
- The abbreviation "SySL" should not be translated.
- Section numbers and structure must remain identical to the English original.
- If a legal concept has no direct equivalent in the target language, use a descriptive phrase and add a translator's note.

### Reviewer Expectations

Translation pull requests should be reviewed by at least one native speaker of the target language. The maintainer may solicit reviews from the community.
