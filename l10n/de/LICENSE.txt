                             PRÄAMBEL

Die Synthetische-Quellcode-Lizenz (SySL) ist für Software konzipiert,
deren Quellcode wesentlich oder vollständig durch künstliche
Intelligenzmodelle generiert wurde. Traditionelle Open-Source-Lizenzen
gehen von einem menschlichen Urheber aus, der das Urheberrecht an dem
Werk innehat. SySL erkennt an, dass KI-generierter Code eine neuartige
rechtliche Stellung einnimmt — eine, in der der Urheberrechtsschutz
ungewiss, nicht vorhanden oder umstritten sein kann — und bietet einen
Rahmen, der unabhängig davon funktioniert.

SySL etabliert drei neue Normen für das KI-Zeitalter:

  1. TRANSPARENZ. Jeder Nutzer und jedes abgeleitete Werk muss
     offenlegen, dass der Code KI-generiert ist und welche Modelle
     verwendet wurden.

  2. RECHTSSICHERHEIT. Die Lizenz fungiert als verbindlicher Vertrag
     unabhängig vom Urheberrecht, sodass Unsicherheit über den
     Urheberrechtsstatus keine Unsicherheit über die zulässige Nutzung
     schafft.

  3. RISIKOANERKENNUNG. KI-generierter Code weist Versagensmodi auf,
     die sich von menschengeschriebenem Code unterscheiden. Nutzer
     akzeptieren dies ausdrücklich.

Diese Normen sind keine Freiheitsbeschränkungen. Sie sind das Fundament
eines neuen Gesellschaftsvertrags: Wir dürfen Software mit KI bauen,
aber wir müssen dazu stehen.

   „Software" bezeichnet den Quellcode, Objektcode, die Dokumentation
   und die dazugehörigen Materialien, die unter dieser Lizenz zur

   „KI-generiert" bezeichnet, dass etwas durch ein künstliches
   Intelligenzmodell oder -system erzeugt wurde, einschließlich
   großer Sprachmodelle, Codegenerierungsmodelle und anderer
   maschineller Lernsysteme.

   „Autor" bezeichnet den oder die Menschen, die den
   KI-Generierungsprozess durch Architekturgestaltung, Prompt
   Engineering, Vorgabenspezifikation, Qualitätssicherung und
   Integration gesteuert haben.

   „Modell" bezeichnet das spezifische KI-System oder die Systeme, die
   zur Generierung der Software verwendet wurden.

   „Sie" bezeichnet die natürliche oder juristische Person, die die von
   dieser Lizenz gewährten Berechtigungen ausübt.

   „Beitrag" bezeichnet jede Änderung, Hinzufügung, Übersetzung,
   Anpassung oder jedes abgeleitete Werk der Software.

   „Abgeleitetes Werk" hat die durch das anwendbare Urheberrecht
   vorgegebene Bedeutung oder, wo das Urheberrecht nicht anwendbar
   ist, jedes Werk, das die Software enthält, verändert oder auf ihr

   „Verbreiten" bezeichnet die Bereitstellung einer Kopie der Software
   oder eines Abgeleiteten Werks an einen Dritten, sei es durch
   physische Medien, Netzwerkübertragung oder Bereitstellung zum
