                                   PRÉAMBULE

La Licence de Source Synthétique (SySL) est conçue pour les logiciels dont le code source est substantiellement ou entièrement généré par des modèles d'intelligence artificielle. Les licences open source traditionnelles présupposent un auteur humain titulaire de droits d'auteur sur l'œuvre. La SySL reconnaît que le code généré par l'IA occupe une position juridique inédite — une position dans laquelle la protection par le droit d'auteur peut être incertaine, absente ou contestée — et fournit un cadre qui fonctionne indépendamment de cette incertitude.

La SySL établit trois normes nouvelles pour l'ère de l'IA :

  1. TRANSPARENCE. Chaque utilisateur et chaque œuvre dérivée doit divulguer que le code a été généré par IA et quels modèles ont été utilisés.

  2. SÉCURITÉ JURIDIQUE. La licence fonctionne comme un contrat contraignant indépendant du droit d'auteur, de sorte que l'incertitude quant au statut du droit d'auteur n'engendre pas d'incertitude quant à l'utilisation autorisée.

  3. RECONNAISSANCE DES RISQUES. Le code généré par l'IA présente des modes de défaillance distincts de ceux du code écrit par un humain. Les utilisateurs l'acceptent explicitement.

Ces normes ne constituent pas des restrictions à la liberté. Elles sont le fondement d'un nouveau contrat social : nous pouvons construire des logiciels avec l'IA, mais nous devons le dire.

   « Vous » désigne la personne physique ou morale exerçant les autorisations accordées par la présente licence.

   « Contribution » désigne toute modification, ajout, traduction, adaptation ou œuvre dérivée du Logiciel.
