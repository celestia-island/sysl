                             PRÄAMBEL

Die Synthetische-Quellcode-Lizenz (SySL) ist für Software konzipiert,
deren Quellcode wesentlich oder vollständig durch künstliche
Intelligenzmodelle generiert wurde. Traditionelle Open-Source-Lizenzen
gehen von einem menschlichen Urheber aus, der das Urheberrecht an dem
Werk innehat. SySL erkennt an, dass KI-generierter Code eine neuartige
rechtliche Stellung einnimmt — eine, in der der Urheberrechtsschutz
ungewiss, nicht vorhanden oder umstritten sein kann — und bietet einen
Rahmen, der unabhängig davon funktioniert.

SySL etabliert drei neue Normen für das KI-Zeitalter:

  1. TRANSPARENZ. Jeder Nutzer und jedes abgeleitete Werk muss
     offenlegen, dass der Code KI-generiert ist und welche Modelle
     verwendet wurden.

  2. RECHTSSICHERHEIT. Die Lizenz fungiert als verbindlicher Vertrag
     unabhängig vom Urheberrecht, sodass Unsicherheit über den
     Urheberrechtsstatus keine Unsicherheit über die zulässige Nutzung
     schafft.

  3. RISIKOANERKENNUNG. KI-generierter Code weist Versagensmodi auf,
     die sich von menschengeschriebenem Code unterscheiden. Nutzer
     akzeptieren dies ausdrücklich.

Diese Normen sind keine Freiheitsbeschränkungen. Sie sind das Fundament
eines neuen Gesellschaftsvertrags: Wir dürfen Software mit KI bauen,
aber wir müssen dazu stehen.
